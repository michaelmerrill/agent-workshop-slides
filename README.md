# Build an Agent with Vercel

![Build an Agent with Vercel](/public/og.png)

Slide deck for the "Build an Agent with Vercel" webinar.

## Overview

Learn how to go from idea to deployed agent in under an hour using AI SDK, Next.js, and Vercel.

### Topics

- **What is an agent?** — definitions, perspectives, and maturity levels
- **Vercel AI cloud primitives** — the platform building blocks
- **AI SDK v6** — structured output, agent abstractions, composable architecture, UI integration, and tool approval
- **Demo** — building an agent live
- **Hands-on: AI Gateway** — get model access with a single API key (BYOK support, observability, cost management)
- **Hands-on: Agent setup** — configure Claude Code and Cursor with the AI Gateway
- **Skills** — what they are, why build them, how to build, deploy, and discover skills
- **Q&A**

## Tech Stack

- [Next.js](https://nextjs.org) 16
- [AI SDK](https://sdk.vercel.ai) v6
- [Convex](https://convex.dev)
- [Tailwind CSS](https://tailwindcss.com) v4
